# Table-of-contents
Automatically detects headings, figures, diagrams, tables, and charts, then assigns appropriate Outline Levels while preserving the original formatting.
# Usage
**English:*
1. Open **Microsoft Word**.
2. Press **Alt + F11** to open the VBA Editor.
3. Select **Insert → Module**.
4. Open the **AutoOutlineLevel** file, copy all the code, and paste it into the new module.
5. Press **Ctrl + S**, then close the VBA Editor.
6. Reopen the document (if necessary).
7. Press **Alt + F8**, select **Module1.AutoOutlineLevel**, and click **Run**.

**Tiếng Việt:*
1. Mở **Microsoft Word**.
2. Nhấn **Alt + F11** để mở **VBA Editor**.
3. Chọn **Insert → Module**.
4. Mở file **AutoOutlineLevel**, sao chép toàn bộ mã và dán vào Module vừa tạo.
5. Nhấn **Ctrl + S**, đóng VBA Editor, sau đó mở lại tài liệu nếu cần.
6. Nhấn **Alt + F8**, chọn **Module1.AutoOutlineLevel**, rồi nhấn **Run** để chạy macro.
